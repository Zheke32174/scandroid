## Method notes

- Used `curl -I` against the public github.com URL (no API auth).
- Anonymous GitHub API has 60/hr — far too low for 1,835 lookups.
- HEAD on github.com itself is not API-rate-limited but does throttle
  at high concurrency. The first pass (parallelism=30) hit 429 on
  ~778 of 1,835 requests; a sequential retry pass (parallelism=1,
  1s sleep between) cleared most of those.
- Status codes that may appear: 200=alive, 301=moved, 404=gone,
  451=legally removed, 429=rate-limited (incomplete signal).
